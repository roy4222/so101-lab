## 標籤索引

每篇日誌使用 1–3 個標籤。

| 標籤 | 說明 |
|------|------|
| `#hardware` | 組裝、接線、馬達、機械結構 |
| `#calibration` | 校正相關 |
| `#teleop` | 遙操作（Leader-Follower、手把、VR） |
| `#camera` | 攝影機設定與串流（含 RealSense） |
| `#dataset` | 資料集錄製、管理、品質 |
| `#training` | 模型訓練（ACT、Diffusion、VLA） |
| `#deployment` | 推論部署（本地、Jetson、非同步） |
| `#dual-arm` | 雙臂相關 |
| `#simulation` | 模擬環境（Isaac Sim、MuJoCo） |
| `#mobile` | 移動底盤、導航 |
| `#debug` | 除錯、踩坑 |
| `#setup` | 環境安裝、依賴管理 |

完整標籤定義請參閱 [Spec 文件](../superpowers/specs/2026-03-27-docs-structure-design.md)。
