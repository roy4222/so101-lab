# Phase 08 — 移動機器人

## 1. 目標

將雙臂系統裝上輪式底盤，整合導航功能，實現移動 + 操作的端到端任務。最終形態對齊 XLeRobot 平台。

## 2. 前置條件

- [Phase 06 — 雙臂改造](./06-dual-arm.md) 已完成
- 移動底盤零件（RASKOG 推車、輪組、馬達）
- 導航感測器
- 參考 [硬體清單](../references/hardware.md)

## 3. 輸入 / 輸出

| 項目 | 說明 |
|------|------|
| **輸入** | 已驗證的雙臂系統、移動底盤硬體 |
| **輸出** | 可移動的雙臂機器人、導航系統、端到端任務 demo |

## 4. 步驟

> **Phase 6 完成後補充詳細步驟。**

預期方向：

- **硬體設計**：參考 XLeRobot 硬體設計（v0.4.0）
- **輪組方案**：三輪全向 vs 雙輪差速
- **機械組裝**：RASKOG 推車改裝
- **移動底盤控制**：ZMQ client-host 架構
- **導航**：LIDAR / RGBD SLAM
- **機上電腦**：Jetson Orin Nano
- **VLA 部署**：SmolVLA / Pi0
- **端到端任務**：移動 → 定位 → 操作 → 完成

## 5. 驗證方式

- 自主移動到指定位置
- 移動後可執行操作任務
- 系統持續運行 30 分鐘以上

## 10. 風險 / Blockers

| 風險 | 說明 | 緩解措施 |
|------|------|----------|
| 導航穩定性 | 室內導航可能受環境影響 | 多場景測試，調整 SLAM 參數 |
| RASKOG 承重和震動 | 推車承重和行進中震動影響操作 | 加裝減震、確認承重餘量 |
| 電源續航 | 移動 + 運算 + 馬達耗電量大 | 提前計算功耗，準備足夠容量電池 |
| 整合複雜度 | 移動 + 雙臂 + 視覺 + 導航的系統整合 | 分模組測試，逐步整合 |
